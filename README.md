# Boopee
읽은 책에 대한 메모를 남기고 사람들과 공유하는 iOS 앱

- 팀원
    - 1인 개발
- 진행 기간
    - 기획: 2024.03.03
    - 개발: 2024.03.03 ~ 진행중

## 개발 환경 및 기술
- 개발 환경
    - Swift 5, xcode 15.0
    - 라이트/다크모드 지원, 가로모드 미지원
- Deployment Target
    - iOS 15.0
- 라이브러리
    - RxSwift, RxAlamofire, Kingfisher, Snapkit
- API
    - Kakao REST API

## 주요 화면 및 기능
- `탐색`: 사람들이 남긴 메모를 탐색할 수 있습니다.
- `작성`: 책을 검색하고 메모를 작성할 수 있습니다.
- `마이페이지`: 내가 작성한 메모를 관리할 수 있습니다.
